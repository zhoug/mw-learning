## 拆书 Skill 是什么

摸宝开源的 Book-learning-skill，核心思路是：

> 用 AI 帮你把一本书拆解、重组、内化，而不是简单摘要。

## 四步流程

```
想法 → 测试 → 环境准备 → Codex 实现 → 迭代
```

## 我的理解

这个 Skill 的精髓不在于「读得更快」，而在于：

1. **结构化输出** — 不是散乱的笔记，而是分层知识
2. **AI 作为教练** — 提问、引导，而非替代思考
3. **知识可检索** — 存入 Obsidian，形成可查询的知识库

## LLM Wiki 三层结构

摸宝的知识库方法论：

```
Raw（原始） → Wiki（整理） → Schema（结构化）
```

每一层都是对上一层的精炼和抽象，最终形成「可被 AI 理解」的知识。
